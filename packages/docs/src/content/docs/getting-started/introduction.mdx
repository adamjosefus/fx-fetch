A production-ready solution for safe & simple HTTP fetching built with
[EffectTS](https://effect.website/). **Designed with great developer
experience in mind.**




## Modules Overview

- [Request](./01-request.md) — Constructors and helpers for Http requests
- [Fetch](./02-fetch.md) — Services and functions for performing Http requests
- [Response](./03-response.md) — Constructors and utilities for handling Http responses
- [Url](./04-url.md) — Constructors and helpers for Url manipulation.
