From the last minor [release (1.0)](../1-0), many things have happened.

This release includes new features and many small fixes and refactors that improve the library's overall quality.
We took a huge step toward making Fx-Fetch much more agnostic to the runtime environment by making the tsconfig DOM agnostic.
